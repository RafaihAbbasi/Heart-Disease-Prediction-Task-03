# Heart Disease Prediction - Task 03

## Project Overview
This project was developed as part of my AI/ML internship task. The goal of this project is to predict whether a person is likely to have heart disease using machine learning techniques and health-related data.

The project follows a complete machine learning workflow including data preprocessing, model training, evaluation, and result analysis.

---

## Objectives
- Understand and explore the dataset
- Clean and preprocess the data
- Train machine learning models
- Compare model performance
- Evaluate results using different metrics
- Identify important features affecting prediction

---

## Dataset Information
The dataset contains medical and health-related information of patients, such as:

- Age
- Sex
- Blood Pressure
- Cholesterol Level
- Heart Rate
- Chest Pain Type
- Blood Sugar
- ECG Results
- Other medical indicators

Target Variable:
- Heart Disease Presence (Yes/No)

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

---

## Project Workflow

### 1. Data Loading
The dataset was loaded and inspected to understand its structure.

### 2. Data Preprocessing
The following preprocessing steps were performed:

- Handling missing values
- Encoding categorical features
- Feature selection
- Data cleaning

### 3. Exploratory Data Analysis (EDA)
Different visualizations and analysis techniques were used to understand the data patterns.

### 4. Model Training
Machine learning models were trained on the processed dataset.

Models used:

- Logistic Regression
- Decision Tree Classifier

### 5. Model Evaluation
Models were evaluated using:

- Accuracy Score
- Confusion Matrix
- ROC-AUC Score
- Classification Metrics

---

## Results

The trained models were able to learn patterns from the dataset and predict heart disease with good performance. Model comparison was performed to determine which algorithm produced better results.
